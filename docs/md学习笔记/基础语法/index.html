基础语法

https://www.itbaima.cn/document/o0ab271mkdsas87

段落和换行​

  • 换行效果： 在上一行的末尾连续添加两个空格，然后再换行。
  • 换段效果： 连续换两行。

加粗和斜体​

想要使得文本加粗，我们只需要在需要加粗文本两边添加两个*星号或_下划线:

这是一段文本，我是需要加粗的内容请帮我加一下粗
这是一段文本，__我是需要加粗的内容__请帮我加一下粗

斜体与加粗类似，我们只需在需要斜体的文本两边添加一个*星号或_下划线:

这是一段文本，我是需要斜体的内容请帮我倾斜
这是一段文本，_我是需要斜体的内容_请帮我倾斜

如果需要同时使用加粗和斜体，我们可以直接使用连续的三个*星号或是_下划线:

这是一段文本，我是同时加粗和斜体的内容

由于部分编辑器在使用_下划线时无法实现加粗效果，因此推荐使用*号。

删除线​

想要为文本添加删除线，我们只需要在需要加删除线的文本两边添加两个~波浪线即可：

这是一段错误的文本，而我是需要删除的内容

分割线​

分割线的创建非常简单，只需在单独一行使用连续三个及以上的*星号或_下划线:

这是分割线上的文本

这是分割线下的文本

标题​

标题分为多种等级，不同的等级表示不同级别下的标题，我们可以通过连续添加多个#井号来表示多个不同等级的标题，但是最多只支持到六级标题:

 # 这是一级标题
 ## 这是二级标题
 ### 这是三级标题
 #### 这是四级标题
 ##### 这是五级标题
 ###### 这是六级标题(最多支持六级标题)

除了使用#井号创建之外，一级标题和二级标题也可以使用其他符号快速创建，一级标题可以使用=等号快速创建，二级标题可以使用-减号快速创建，都只需要在标题文本的下方添加即可：

这是一级标题
=
***
这是二级标题
-

列表和勾选框​

要创建一个无序列表，我们只需要使用*星号作为前缀，注意需要空格:

  • 我是列表第一项
  • 我是列表第二项
  • 我是列表第三项

除了星号之外，我们也可以使用加号 + 或是减号 -来代替，效果完全相同。注意，这三种符号在一个列表中只能统一使用同一种，不能混用，如果出现混用的情况，会被认为是上一个列表下的一个新的列表，比如下面就被认为是三个列表了：

  • 我是列表第一项
  • 我是列表第二项
  • 我是列表第三项

要创建一个带数字的有序列表，我们可以使用数字的123来表示，首先输入数字然后添加.小数点作为前缀，注意添加空格：

  1. 我是列表第一项
  2. 我是列表第二项
  3. 我是列表第三项

列表支持多个层级嵌套，我们可以在一级列表下创建二级列表，要表示二级列表，我们可以在前面添加两个或四个空格（也可以是一个TAB缩进），表示此列表是上一个列表的子级：

  • 我是一级列表
    • 我是二级列表
      • 我是三级列表

将列表变为勾选框，只需要添加一个方括号即可：

  • 我没有被选中
  • 我被选中了

在方括号中间填写空格表示未选中，填写大写或小写的x表示已选中。

代码块​

使用三个连续的 ` 符号来添加代码块，注意前后都需要添加，囊括起来：

<body>
    <p id="x">我是p标签</p>
    <button onclick="an()">修改p标签</button>
    <script>
        function an() {
            let x = document.getElementById('x')
            x.innerText = "hello word"
        }
    </script>
</body>

在上面的三个 ` 后可以添加语言名称，比如我们这里添加的是js代码，那就添加js名称上去，表示这个代码块是一段js代码，便于Markdown编辑器分辨代码并为其添加相应的高亮展示。

注意： 代码块中的所有内容都会被认为是代码，不可以嵌套其他Markdown语法。

除了编写多行代码之外，如果只是加单编写一个行内展示的代码，比如方法名称之类的，我们可以使用单个 ` 囊括行内代码块部分：

数组的常用方法有map(),filter(),find(),reduce()等等

引用​

当我们引用他人文章内容时，往往需要特别标注此部分，而Markdown为我们提供了更加方便的文本引用块，我们只需要在最前面添加>大于符号即可表示后面的文本为引用文本，注意空格。

这是我引用的内容

被引用的文本会被一个特殊的块包裹起来，类似于代码块，在引用块内换行同样需要像普通文本那样，如果只是单纯换行编写，会被认为是一个空格。

这是一段文本 看看我换行了吗
记得加两个空格才能换行

对于多行内容的引用，推荐每一行的前面都添加>大于符号，效果与上面是完全一样的，并且看上去会更加直观：

方便操作
更直观
推荐使用

在引用块内，我们也可以嵌套其他Markdown内容，比如列表或是代码块，引用还能继续嵌套引用，甚至很多层，只需要连续添加>大于符号前缀即可，注意空格:

我是最外层的引用

我是中间层的引用

我是最内层的引用
